# Checkout-Pro-Antigravity

A premium Darts Checkout Calculator built with speed and aesthetics in mind.

## Features
- Optimal checkout paths from 170.
- Modern, responsive UI.
- Offline-ready logic.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser or run a local dev server.
3. (Optional) Set up environment variables in `.env`.
